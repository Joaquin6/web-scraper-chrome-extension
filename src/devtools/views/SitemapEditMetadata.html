Поддерживаемые шаблоны URL:
1. Числовой с опциональным шагом и дозаполнением нулями – [START-END:STEP] – [001-010:10]
2. На основе дат – [date<PATTERN><START><END>] – [date<dd.MM.yyyy><01.01.2017><now>]
    вместо дат можно использовать yesterday(вчера) / now(сейчас) / tomorrow(завтра)
    остальные шаблоны (в стиле Java)
      yyyy полный год (4 digits)
      yy последние 2 цифры года
      MMM   Jan, Feb, Mar, Apr, May, Jun, Jul, Aug, Sep, Oct, Nov, Dec
      MM номер месяца (01-12)
      dd день месяца

Вы можете указать модель карты обхода в виде листа JSON объектов.
Пример: [ { "entity": "Имя сущности", "field": "Поле сущности", "field_name": "Ожидаемый идентификатор поля" } ]
